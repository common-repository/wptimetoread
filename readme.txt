Reading time.

== Description ==

Simply install. Displays how long a post or page will take to read based not only on the number of words but difficulty.
== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/wptimetoread` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress

That's it!
